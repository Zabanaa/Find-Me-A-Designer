What The Func Starter Kit

This is a boilerplate for you to use on your front-end projects.

It comes with Babel already configured so you can write and enjoy the newest Javascript features and goodies. Thanks to gulp, sass is already installed and will compile all of your stylesheets into one compressed file to increase performance and load time. You will also enjoy automatic browser reloading with BrowserSync, and finally. This starter kit supports JSX, so if you're a React JS dev, you will be able to use it with no problems at all.
